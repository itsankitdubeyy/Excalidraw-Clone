🎯 Overview

✨ A modern Excalidraw Clone built for creating diagrams, sketches, and ideas on a smooth digital canvas.
It provides a minimal, hand-drawn UI experience with powerful drawing tools.

🌐 Live Demo

✨ Features

✔️ Infinite Canvas
✔️ Freehand Drawing ✍️
✔️ Shapes (Rectangle, Circle, Arrow, Line)
✔️ Undo / Redo
✔️ Zoom & Pan
✔️ Export (PNG / JSON)
✔️ Dark Mode 🌙
✔️ Smooth Performance ⚡
